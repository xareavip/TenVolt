Ubuntu Linux 基础命令复习训练手册
训练说明
全程在当前用户家目录下操作，文件 / 目录一次性创建，后续统一复用，不再重复新建
严格按章节顺序完成，所有操作需独立完成，不提供命令答案
遇到命令错误、路径错误需自行排查修正
最终完成后按要求清理环境
任务 1：登录与初始环境确认
远程登录系统（本地终端可跳过）
确认当前所在的绝对路径
清空无关目录与文件，保证操作环境干净
任务 2：目录管理（创建、切换、查看、删除）
创建单个目录
递归创建多级嵌套目录
按要求切换目录：进入目录、返回上一级、回到家目录、返回上一次所在目录
用不同参数查看目录内容：
基础查看
长格式详细查看
人性化大小显示
查看隐藏文件
删除空目录
删除非空目录（含子目录）
验证删除结果
任务 3：文件创建与管理（一次性创建，全程复用）
创建两个文本文件（后续所有操作复用）
新建一个目录，用于统一存放文件
编辑两个文件，写入指定内容
尝试安装文本编辑工具（如遇锁冲突，自行处理）
完成文件复制、重命名、移动操作
删除指定文件
恢复核心文件，确保后续可继续使用
任务 4：文件内容查看、合并与重定向
分别查看两个文件的完整内容
将两个文件内容合并，输出为新文件
用不同方式查看合并后的文件：
分页查看
查看前 N 行
查看后 N 行
实时查看系统日志
使用输出重定向：
把目录查看结果覆盖写入文件
向文件追加新内容
验证文件最终内容
任务 5：系统信息查看
查看系统内核、版本、架构信息
查看磁盘挂载与使用情况
查看内存与交换分区使用情况
查看系统运行时间、登录用户数
查看指定目录总大小
查看当前目录下一级子目录 / 文件大小
任务 6：进程管理
动态查看系统进程
安装增强版进程查看工具并启动
查看系统所有进程
按名称过滤指定进程
按进程名称终止进程
验证进程是否已停止
任务 7：打包与压缩
将指定目录打包为归档文件
对单个文件进行压缩、解压
将指定目录递归压缩为 zip 格式
查看生成的打包 / 压缩文件
任务 8：文件查找与内容检索
在指定范围内按文件名查找文件
递归查找目录中包含指定内容的文件
忽略大小写查找指定内容
查找系统命令所在路径
查找程序安装相关路径
任务 9：环境清理
返回用户家目录
删除本次训练创建的所有目录、文件
确认目录干净，恢复初始状态
训练要求
不查询答案，独立完成所有操作
记录操作中出现的错误及解决方法
命令格式规范，无语法错误
按流程执行，不跳过、不重复创建基础文件
